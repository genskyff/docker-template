# Rsbuild Project

## Setup

Install the dependencies:

```bash
bun install
```

## Get Started

Start the dev server:

```shell
bun run dev
```

Build the app for production:

```shell
bun run build
```

Preview the production build locally:

```shell
bun run preview
```

Lint the code:

```shell
bun run check
```

Fix linting issues:

```shell
bun run fix
```
