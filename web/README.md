# Rsbuild Project

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```shell
pnpm dev
```

Build the app for production:

```shell
pnpm build
```

Preview the production build locally:

```shell
pnpm preview
```

## Docker Compose

To build and run the project using Docker Compose, use the following command:

```shell
docker compose up -d
```
